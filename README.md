# Vanilla JS Project

Це мій перший проект з **Vanilla JS** та **Vite**.

Використано **npm пакет lodash** для генерації випадкових чисел.

## Структура проекту

- `vanilla-app/`
  - `node_modules/` ← встановлені npm пакети
  - `src/` ← код проекту (main.js, style.css)
  - `index.html` ← HTML-шаблон
  - `package.json` ← залежності та скрипти
  - `vite.config.js` ← конфігурація Vite

## Як запустити проект

1. Встановити залежності:
   npm install

2. Запустити сервер Vite:
   npm run dev

3. Відкрити браузер за адресою, яку покаже Vite, зазвичай:
   http://localhost:5173/

## Функціонал

- Відображає заголовок і текст на сторінці.
- Генерує випадкове число від 1 до 100 за допомогою lodash і виводить його на сторінку та у консоль браузера.

## Technologies used

- JavaScript (ES6)
- Vite
- Lodash (npm package)
- HTML & CSS
